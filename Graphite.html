Справка

Поддерживаются базовые арифметические операции (сложение, вычитание, умножение, деление, возведение в степень), число Эйлера (латинская "e"), число Пи (Pi), запись чисел в научной форме (AeB, что равно A*10^B).

Поддерживаются функции, их аргумент необходимо писать в круглых скобках.

Список поддерживаемых функций (X - любое допустимое математическое выражение):

  • sin(X).
    Синус угла X
  • cos(X).
    Косинус угла X
  • tg(X), tan(X).
    Тангенс угла X
  • ctg(X), cot(X).
    Котангенс угла X
  • sec(X).
    Секанс угла X
  • cosec(X), csc(X).
    Косеканс угла X
  • arcsin(X), asin(X).
    Арксинус числа X
  • arccos(X), acos(X).
    Арккосинус числа X
  • arctg(X), atan(X).
    Арктангенс числа X
  • arcctg(X), acot(X).
    Арккотангенс числа X
  • arcsec(X), asec(X).
    Арксеканс числа X
  • arccosec(X), acsc(X).
    Арккосеканс числа X
  • sqrt(X)
    Квадратный корень числа X
  • abs(X)
    Модуль числа X
  • ln(X)
    Натуральный логарифм числа X
  • exp(X)
    Экспонента числа X
  • rtX(Y), rootX(Y)
    Корень числа Y по основанию X
  • rand(X)
    Случайное дробное число от нуля до X
